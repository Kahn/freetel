Layers produced by EAGLE-4.16 and Sunstone-EagleCam (E2 2-LAYER) routines.

	.drt	- drill table
	.L1	- gerber top layer
	.L2	- gerbrer +V layer
	.L3	- gerber GND layer
	.L4	- gerber bot layer
	.slk	- gerber top silkscreen layer
	.smb	- gerber bottom solder mask layer
	.smt	- gerber top solder mask layer
	.tps	- gerber top paste layer

	Finish - Electroless Nickle Immersion Gold
